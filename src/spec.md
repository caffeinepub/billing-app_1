# Specification

## Summary
**Goal:** Build an invoice generation application with multi-currency support, PDF download, and shareable invoices.

**Planned changes:**
- Create backend data model for invoices with customer details, line items, currency codes (USD, INR, etc.), tax calculations, and status tracking
- Implement backend CRUD operations (create, retrieve, list, update, delete invoices)
- Build frontend form for creating invoices with dynamic line items, currency selector, and automatic total calculations
- Display professional invoice preview with correct currency symbols
- Add PDF download functionality for invoices
- Implement shareable link generation for invoices
- Create dashboard to list, view, edit, and delete all invoices
- Apply professional business-themed visual design with trust-conveying color palette

**User-visible outcome:** Users can create professional invoices for any business transaction, select from multiple currencies (USD, INR, etc.), see live calculations with tax, preview invoices in a formatted layout, download them as PDFs, share via links, and manage all invoices from a dashboard.
